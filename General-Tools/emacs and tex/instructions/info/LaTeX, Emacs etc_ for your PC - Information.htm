Emacs information

  • In Emacs, start info mode by pressing "C-h i". For a tutorial, press "C-h t".
  • NT emacs, NTemacs mailinglist, Seach mailinglist
  • Emacs reference card, (a short summary of keystrokes) is distributed with emacs as c:\emacs\etc\refcard.ps.
  • GNU Emacs Manual
  • GNU Emacs Lisp Reference Manual
  • http://www.emacs.org/

AUC-TeX information

  • In Emacs, press "C-h i" and choose auctex. (First, you need to do the step "Info manuals for AUCTeX" in the tuning page.)
  • AucTeX homepage
  • Some sample-files for LaTeX-users who work with auctex. For win32 Xemacs.
